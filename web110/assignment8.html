SQL SELECT Statement

  • SELECT * FROM Customers;

SQL WHERE Clause

  • SELECT * FROM Customers WHERE Country='USA';

SQL COUNT() Function

  • SELECT COUNT(*) FROM Customers;

SQL INSERT INTO Statement

  • INSERT INTO Customers (CustomerName, ContactName, Address, City, PostalCode, Country) VALUES ('Cardinal', 'Tom B. Erichsen', 'Skagen 21', 'Stavanger', '4006', 'Norway');

SQL UPDATE Statement

  • UPDATE Customers SET ContactName='Alfred Schmidt', City='Frankfurt' WHERE CustomerID=1;

I learned that SQL is used to work with information stored in a database. The SELECT statement displays data, the WHERE clause filters results, COUNT() counts records, INSERT INTO adds new information, and UPDATE changes existing information. These activities helped me understand how SQL is used to manage data. I can use these skills in future web development projects, college classes, and jobs that use databases.
