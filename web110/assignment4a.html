Activity 1: Internet Reference Models In this activity, I learned about Internet Reference Models and how computers communicate over a network. I learned that the TCP/IP model is made up of different layers, and each layer has a specific job. These layers work together to send and receive information across the internet. Before this lesson, I did not realize how many steps are involved when data travels from one device to another. Learning about the different layers helped me understand how websites, emails, and other online services work. It also showed me how networks are organized so that devices can communicate efficiently and reliably. I can apply this knowledge in my current and future environment by having a better understanding of how computer networks function. If I work with technology in the future, this information will help me understand networking concepts more easily. It can also help me identify where communication problems may happen within a network. Even as a regular computer user, knowing how data moves through a network helps me better understand the technology I use every day. Activity 2: Fix Wi-Fi Connection issues and windows In this activity, I learned about common Wi-Fi connection problems and ways to troubleshoot them. I learned that internet issues can be caused by different things, such as incorrect settings, software problems, outdated drivers, or issues with the router. The lesson showed the importance of checking possible causes one step at a time instead of guessing. I also learned that Windows includes tools that can help find and fix network problems. This activity helped me understand that troubleshooting is a process that requires patience and logical thinking. I can use this knowledge in my current environment whenever I experience internet connection problems at home or on personal devices. Instead of immediately asking for help, I can use troubleshooting techniques to find possible solutions. In a future workplace, these skills could help me assist coworkers or customers who are having network issues. Since internet access is important for communication, work, and learning, understanding basic Wi-Fi troubleshooting can save time and help keep things running smoothly.
