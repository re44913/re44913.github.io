In this lesson, I learned that CSS stylesheets are used to change the way a website looks without changing the HTML content. I learned that external stylesheets help keep the same design on every page, which makes websites easier to update and manage. I also learned about internal and inline styles and how each one can be used for different purposes. CSS can change colors, fonts, text size, spacing, and the background of a webpage to make it look more organized and easier to read. I can use these skills in future school assignments, personal websites, or even work projects to create webpages that look clean, consistent, and more professional.
