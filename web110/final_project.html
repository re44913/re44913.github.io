Final Project: Web Accessibility

This semester I learned many important skills about the Internet and web development. I learned how to build webpages using HTML and style them with CSS. I also learned how to publish my work using GitHub and check my code with HTML and CSS validators. One of the biggest lessons I learned was the importance of web accessibility. I now understand how features like alt text, headings, keyboard navigation, and good color contrast help make websites easier for everyone to use. This course has given me a strong foundation in web development, and I will continue using these skills in future classes and future projects.
